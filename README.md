# EntrLabs Frameworks

**Research-backed systems for growth through service.**

EntrLabs translates leadership research, learning science, and organizational theory into practical frameworks for people, teams, and institutions.

This repository is the public framework library.

## Core Doctrine

```text
Strength Through Service
```

EntrLabs is built on the belief that people and institutions grow best when development is connected to contribution.

## Framework Map

| Framework | Use Case | Primary Audience |
| --- | --- | --- |
| Service Leadership | Central doctrine | Leaders, educators, founders, institutions |
| String Leadership Theory | Leadership development | Managers, founders, professionals |
| EntrSL / ELEM | Entrepreneurial learning engagement | Researchers, educators, universities |
| HYOH | Career agency | Students, early professionals, career builders |
| Culture Snapshot | Recognition and culture review | Teams, SMEs, nonprofits, institutions |

## What EntrLabs Builds

### Frameworks

Original models for leadership, service, culture, learning, and career agency.

### Tools

Worksheets, diagnostics, field guides, and prompts that make research usable.

### Systems

Implementation support for teams, educators, founders, and institutions.

## Guiding Thesis

The future of leadership is not personality-driven. It is system-driven, service-centered, and research-backed.
